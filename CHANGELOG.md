# Changelog

## [1.0.0] - 2024-03-23

### 核心功能

#### 文章管理
- ✅ 文章创建、编辑、删除功能
- ✅ 富文本编辑器（基于 SunEditor）
- ✅ Markdown 和 HTML 双格式支持
- ✅ 文章分类和标签管理
- ✅ 文章状态管理（草稿/已发布）
- ✅ 文章摘要自动生成

#### 媒体库
- ✅ 图片上传和预览
- ✅ 剪贴板粘贴图片
- ✅ 媒体文件管理
- ✅ 图片 URL 复制

#### 本地文件系统
- ✅ File System Access API 集成
- ✅ 本地文件夹选择授权
- ✅ 文章本地存储（JSON + Markdown/HTML）
- ✅ 媒体文件本地存储
- ✅ 权限验证和自动重连

#### 多仓库架构
- ✅ 本地存储支持
- ✅ GitHub 配置界面
- ✅ Gitee 配置界面
- ✅ 仓库切换功能
- ✅ 连接状态管理

#### 主题系统
- ✅ 9种配色方案（purple, blue, green, orange, pink, cyan, red, indigo）
- ✅ 深色/浅色模式切换
- ✅ CSS 变量动态生成
- ✅ 主题持久化存储

#### 控制台界面
- ✅ 概览面板（统计卡片、分类标签统计）
- ✅ 文章列表（卡片/列表视图、搜索筛选）
- ✅ 分类标签管理
- ✅ 设置页面（主题、仓库、AI、个人信息）

### 技术栈

- **框架**: Nuxt 4 (Vue 3)
- **语言**: TypeScript
- **UI**: Element Plus + Tailwind CSS
- **状态管理**: Pinia
- **编辑器**: SunEditor
- **存储**: File System Access API + localStorage

### 已知限制

- AI 功能仅为占位符，未实际实现
- GitHub/Gitee 仅为配置界面，无实际 API 集成
- 编辑器编辑文章时数据加载存在问题
- 部分类型定义使用 `any`，类型安全有待提升

## [1.1.0] - 计划中

### 核心功能增强

#### AI 写作助手
- AI 续写功能
- 智能润色
- 自动生成摘要
- 支持 Ollama 本地 LLM
- 支持 OpenAI/Claude API

#### GitHub 集成
- OAuth 授权流程
- 仓库列表获取
- 文章同步（拉取/推送）
- 媒体文件 CDN 链接

#### 编辑器改进
- 修复文章数据加载
- 更好的错误处理
- 自动保存草稿

### 稳定性提升

#### 错误处理
- 全局错误边界
- 友好的错误提示
- 重试机制

#### 类型安全
- 移除所有 `any` 类型

#### 性能优化
- 虚拟滚动

#### SEO 工具
